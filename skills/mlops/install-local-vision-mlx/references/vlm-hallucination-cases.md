# VLM 幻觉案例库

记录 Qwen3-VL 实际在使用中出现的识别错误，给未来 session 校准期望。
**用法**：被用户纠正过的案例都加进来。

---

## 案例 1：长椅扶手→"棕色泰迪熊" (2026-06-24)

**图**：两个扎小辫的小女孩坐户外木长椅，左边女孩前倾低头，双手抱住自己身体

**真实情况**：左边女孩只是抱住自己难过，**画面里没有玩偶/任何熊**

**模型第一次回答**：
> "她正低着头，身体前倾，将脸埋在双臂之间……她的双手紧紧抱着一个物体……物体的主体颜色是棕色，看起来是泥土或木头的颜色……"

**用引导性提问追问后（"只描述真实看到的"）**：
> "左边女孩抱着一个棕色的、看起来像木制或布艺的玩偶。"

**真相**（用户告知）：**"棕色块状是长椅的边"**——女孩前倾的姿势把长椅的木质扶手挡在了怀里区域，模型把它脑补成了"被抱着的玩偶"。

**agent 当时犯的二次错误**：
1. 信任了模型的描述
2. 给用户写了一整段"小女孩抱熊熊找安慰、另一个孩子默默陪伴"的情感分析
3. 整段都建立在不存在的玩偶上

**规律提取**：
- 触发条件：孩子 + 难过姿势 + 怀里有任何棕色形状 → 模型→玩偶
- 模型对**情绪场景中"应该出现的物品"**有补全偏置
- 即使用"只描述真实看到的，不要猜测"的提示词，模型仍可能在棕色物体识别上偏向"玩偶"

## 案例 2：CBD 建筑识别幻觉 (2026-06-24)

**图**：从高楼俯拍的某金融区，多栋摩天楼 + 主干道，远处一栋白色圆顶建筑

**模型回答**：
- 第一次："这是上海陆家嘴，能看到上海中心大厦螺旋形、东方明珠、IFC 圆顶……能看到'中国银行'蓝色字样"
- 追问"只报告实际看到的字"："7 栋楼都无可见文字，只有 2 号楼顶有模糊蓝色标识，看不清"

**真相**：模型第一次的"中国银行字样"是脑补的。它对**著名 CBD 鸟瞰图**有强烈的"上海陆家嘴模板"补全偏置。

**规律提取**：
- 触发条件：高楼俯拍 + 摩天楼 + 主干道 → 模型→"上海陆家嘴"
- 模型把**最有名的同类场景**当作识别模板套上去，连带补全地标名字
- 用"只报告你看到的字"重问，模型会**自己撤回**幻觉

## 案例 3：合成图识别（正确案例） (2026-06-24)

**图**：抖音号 2010887186 制作的"世界奇观叠拼"——上海陆家嘴/布达拉宫/金字塔/自由女神叠加

**模型回答**：准确识别出"这是数字合成/AI 生成图，不是真实照片"，并且大致列出了每一层的内容。**抖音号水印识别正确**。

**为什么这个案例对**：图本身就是公开传播的合成图，模型的训练数据里见过类似的拼贴模式；水印是清晰的小字，OCR 路径走通。

## 案例 4：股票界面识别（正确案例） (2026-06-23)

**图**：手机炒股 App 的纳指期货 CFD 界面

**模型识别正确的细节**：
- 手机时间 5:56、网络速度 0.08 KB/s、电量 49%、5G
- 价格 29734.53、涨跌 -918.97、-3.00%
- 今开 30662.75、最高 30701.25、最低 29577.25
- MACD 参数 [12,26,9]、DIFF 6.466、DEA 4.544
- 底部广告"全油万雪！观望错过，开户正当当时！"
- 新浪财经水印

**为什么这个案例对**：图里全是**清晰的数字/中文字符**，走 OCR 路径；模型对界面元素（k 线、按钮、广告条）的结构化识别也成熟。

## 总结：什么时候信，什么时候不信

| 任务类型 | 信任度 | 注意 |
|---|---|---|
| 屏幕截图 OCR / 数字识别 | ✅ 高 | 用得着 |
| 文档 / 表格 / 票据 | ✅ 高 | 用得着 |
| Logo / 招牌 / 品牌文字 | ✅ 中 | 字清晰才行 |
| 界面布局 / UI 元素 | ✅ 中 | 结构化场景 |
| 自然场景物体识别（清晰背景） | ⚠️ 中 | 复述要保守 |
| 自然场景物体识别（遮挡/复杂背景） | ❌ 低 | 必须存疑 |
| 情感 / 关系 / 故事推理 | ❌ 低 | 只能作为线索，不能作为事实 |
| 著名地标场景（鸟瞰 CBD 等） | ❌ 低 | 模型套模板 |
| 抽象艺术 / 合成图 | ✅ 中 | 见过的反而准 |

**最重要的一条**：当用户给你的图你**自己一眼看不太清那个细节**时（比如被遮挡的物体、远处模糊的字），把模型的回答**降级为"模型识别为 X"**而不是"图里有 X"，等用户确认后再加工。

## 防幻觉提示词模板

当怀疑模型可能幻觉时，用这种锁死的提示重问：

```
请只做一件事：从左到右、从近到远，把图中能看清的物体逐一列出。
对每个物体，只说你"实际看到的颜色、形状、位置"。
不要猜测物体是什么、不要根据外形推测用途、不要发挥情感联想。
如果某处看不清，就说"看不清"。
```

或者针对文字：

```
列出你能看到的所有文字、招牌、Logo 上的字。
如果某处字模糊或太远看不清，就说"无可辨认文字"。
不要根据建筑外形或场景推测可能是什么字。
```
